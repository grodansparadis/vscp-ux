VSCP google table demo

This demo uses the table from Google charts to display temperature measurement values in degrees Celsius. It displays (as the other samples) all CLASS1.MEASUREMENT, Type=6/VSCP_TYPE_MEASUREMENT_TEMPERATURE values that comes from sensor=2.

The event.html sample can be used to set temperature values if you don't have live hardware.
