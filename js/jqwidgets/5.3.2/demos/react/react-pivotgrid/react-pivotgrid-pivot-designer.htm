Description

This is an example of the interaction between the React Pivot Grid Component and the React Pivot Grid Designer Component. You can use the pivot grid designer to select which fields will be displayed on rows and columns. You can also select different value fields and choose an aggregation function. Simply drag and drop fields in the respective area and the pivot grid's content will update automatically. You can also click on the menu of each columns, rows, filters or values item and select additional options like text alignment, number formatting and numbers alignment. The pivot grid designer also lets you specify pivot table filters. Drag and drop any field in the filters area, open the menu and uncheck the items you want to filter from the aggregation calculations. The example also demonstrates how to add a custom aggregation function which calculates the variance of a data set and localization of the pivot table designer with a text that displays the name of the function in the aggregation functions menu.
